# IPL Betting App - Тестовое задание SEOCare

Тестовое задание для SEOCare - адаптивная верстка лендинга для IPL Betting App.

## Описание

Проект представляет собой одностраничный лендинг для мобильного приложения ставок на крикет IPL (Indian Premier League).

**Верстка выполнена с pixel-perfect точностью для десктопной версии**, с адаптацией для планшетов и мобильных устройств.

## Технологии

-   **HTML5** - семантическая разметка
-   **SCSS/Sass** - препроцессор для стилей
-   **TypeScript** - типизированный JavaScript
-   **Vite** - сборщик проекта
-   **Swiper.js** - библиотека для слайдеров
-   **CSS Grid & Flexbox** - современные техники верстки

## Структура проекта

```
├── index.html              # Главная HTML страница
├── src/
│   ├── assets/            # Изображения и статические ресурсы
│   ├── styles/            # SCSS стили
│   │   ├── base/         # Базовые стили (reset)
│   │   ├── components/   # Переиспользуемые компоненты
│   │   ├── layout/       # Макетные стили (container)
│   │   ├── mixins/       # SCSS миксины
│   │   ├── sections/     # Стили секций страницы
│   │   ├── variables.scss # Переменные
│   │   └── main.scss      # Главный файл стилей
│   └── ts/
│       └── main.ts        # TypeScript логика
├── package.json
└── README.md
```

## Особенности верстки

### Десктопная версия

-   **Pixel-perfect верстка** - точное соответствие макету для десктопной версии
-   Адаптивная сетка с использованием CSS Grid и Flexbox
-   Оптимизированные изображения с lazy loading
-   Семантическая HTML разметка для SEO
-   Schema.org разметка для отзывов и FAQ

### Адаптивность

-   **Планшеты** (до 1150px): адаптированная верстка с сохранением функциональности
-   **Мобильные** (до 767px): оптимизированный интерфейс с бургер-меню
-   Горизонтальная прокрутка для некоторых секций на мобильных
-   Адаптивные слайдеры с touch-навигацией

## Компоненты

### Слайдеры (Swiper.js)

-   **IPL секция** - слайдер с прогресс-баром и навигацией
-   **Live секция** - слайдер с матчами
-   **Reviews секция** - слайдер с отзывами
-   **TOC секция** - горизонтальная прокрутка для мобильных
-   **App-download секция** - сетка с брендами

### Аккордеоны

-   **Reasons to Download** - причины скачать приложение
-   **How to use** - инструкция по использованию
-   **FAQ** - часто задаваемые вопросы

## Установка и запуск

### Требования

-   Node.js (версия 18+)
-   npm или yarn

### Установка зависимостей

```bash
npm install
```

### Запуск в режиме разработки

```bash
npm run dev
```

### Сборка для продакшена

```bash
npm run build
```

### Превью продакшен сборки

```bash
npm run preview
```

## Оптимизация

-   Минификация CSS и JavaScript в продакшене
-   Lazy loading для изображений (`loading="lazy"`)
-   Оптимизированные шрифты через Google Fonts
-   Использование современных CSS техник (Grid, Flexbox)
-   Модульная структура SCSS с использованием `@use`
-   Удален неиспользуемый код
-   Оптимизированные импорты Swiper модулей
